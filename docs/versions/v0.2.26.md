# v0.2.26

## Changes

- Automatically upgrade an existing short-lived Google token found in browser storage into the app's 30-day server session.
- Persist valid app sessions back to durable browser storage during app startup.
- Bump app version to `0.2.26` and service worker cache to `minha-adega-v23`.

## Validation

- `pnpm --filter @workspace/minha-adega typecheck`
- `pnpm --filter @workspace/api-server typecheck`
- `pnpm --filter @workspace/minha-adega build`
- `pnpm --filter @workspace/api-server build`

## Rollback

- Roll back app deploy to `v0.2.25` if session startup migration causes unexpected login behavior.
