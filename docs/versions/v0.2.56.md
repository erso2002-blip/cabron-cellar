# v0.2.56

## Changes

- Changes the Pro drink-date suggestion from a generic AI estimate to a web-researched suggestion that prioritizes official winery/producer sources, technical sheets, PDFs, and vintage notes.

- Keeps a fallback profile estimate only when no reliable producer or reputable reference source is available.
- Reconciles Mercado Pago `payment` webhook events by fetching the payment, resolving the related local subscription from the external reference or subscription metadata, and refreshing the preapproval status.
- Bump app/API/workspace version to `0.2.56`, native Android/iOS build numbers to `56`, and service worker cache to `minha-adega-v51`.
