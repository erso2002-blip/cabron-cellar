# v0.2.25

## Changes

- Disable paid subscription buttons on the billing screen.
- Replace the Mercado Pago checkout CTA with `Em breve` for Pro Mensal and Pro Anual.
- Add a `Voltar` action to the public billing screen.
- Exchange Google credentials for a 30-day server session, persisted in browser storage, so approved testers do not need to authorize Google on every app reopen.
- Bump app version to `0.2.25` and service worker cache to `minha-adega-v22`.
