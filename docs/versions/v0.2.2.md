# v0.2.2

- Public app version: `0.2.2`
- Status: release candidate pending production deploy.

## Changes

- Added a cellar filter to `Sua Adega`.
- The filter uses the existing `cellarName` field and includes a `Todas as adegas` option.
- Search by name, producer, and grape remains available.
- Grid/list view switching remains unchanged.

## Validation

- `pnpm --filter @workspace/minha-adega typecheck`
- `PORT=3000 BASE_PATH=/ pnpm --filter @workspace/minha-adega build`

## Notes

- No database migration required.
- Wines without `cellarName` remain visible under `Todas as adegas`.
