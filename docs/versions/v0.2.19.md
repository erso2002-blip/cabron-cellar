# v0.2.19

## Changes

- Persist legal acceptance in browser storage after the user checks the login consent box.
- Reuse the saved acceptance on future login screen visits so returning users are not asked to check the terms again on the same client/PWA.
- Bump app version to `0.2.19` and service worker cache to `minha-adega-v17`.

## Validation

- Pending local build/typecheck.
- Pending production validation after deploy.

## Rollback

- Roll back app deploy to `v0.2.18` if legal acceptance persistence causes unexpected login behavior.

## Known Gap

- This release persists acceptance on the client/PWA. Central database-level legal consent audit remains a recommended later improvement if MyCellar needs compliance reporting across devices.
