# v0.2.20

## Changes

- Add server-side closed beta access control through `CLOSED_BETA_ALLOWED_EMAILS`.
- Block Google SSO, e-mail code requests, e-mail code verification, and existing e-mail sessions when the account e-mail is outside the allowed beta list.
- Send public unauthenticated traffic from `/` to the waitlist while keeping `/login` available for pre-approved testers.
- Bump app version to `0.2.20` and service worker cache to `minha-adega-v18`.
