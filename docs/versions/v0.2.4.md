# v0.2.4

- Public app version: `0.2.4`
- Status: release candidate pending production deploy.

## Changes

- Updated the PWA service worker cache from `minha-adega-v1` to `minha-adega-v2`.
- Removed the root HTML from precache to reduce stale shell risk after deploys.
- Changed SPA navigation handling to network-first, with cached shell only as fallback.
- Added controlled reload when a new service worker takes control.

## Validation

- `pnpm --filter @workspace/minha-adega typecheck`
- `PORT=3000 BASE_PATH=/ pnpm --filter @workspace/minha-adega build`

## Notes

- No database migration required.
- Hotfix targets blank-screen risk on iPhone/PWA caused by stale cached HTML pointing to old assets.
