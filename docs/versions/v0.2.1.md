# v0.2.1

- Public app version: `0.2.1`
- Status: release candidate pending production deploy.

## Changes

- Added a safe "Voltar para adega" action in consumption history.
- Restoring a consumption now increments wine stock and removes the history record in one transaction.
- Label scan now builds a fallback wine name from producer, grape, and vintage when the AI cannot isolate a name.
- Label scan can return `wineryWebsiteUrl` only for official producer/winery domains and avoids retailer or marketplace URLs.
