# v0.2.5

- Public app version: `0.2.5`
- Release type: hotfix
- Date: 2026-06-17

## Summary

- Updated the PWA service worker cache from `minha-adega-v2` to `minha-adega-v3`.
- Added a one-time client navigation on service worker activation so stale iPhone/Safari clients can leave cached `v0.2.0` HTML.

## Validation

- `pnpm --filter @workspace/minha-adega typecheck`
- `PORT=3000 BASE_PATH=/ pnpm --filter @workspace/minha-adega build`

## Risk

- Existing open tabs may reload once after the new service worker activates.
- No database, authentication, or API behavior changed.

## Rollback

- Use Vercel rollback to `v0.2.4` if the forced refresh causes unexpected navigation behavior.
