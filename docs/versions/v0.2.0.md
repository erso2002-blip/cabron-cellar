## Included Changes

- Adds public `/waitlist` page and `/api/waitlist` lead capture endpoint.
- Adds optional winery website field for wine create/edit/detail.
- Adds passwordless email login with six-digit code and database-backed sessions.
- Keeps Google SSO available as the primary login method.

## Required Production Setup

- Apply database schema for:
  - `waitlist_leads`
  - `email_login_codes`
  - `email_sessions`
  - `wines.winery_website_url`
- Configure email delivery before relying on email login:
  - `RESEND_API_KEY`
  - optional `AUTH_EMAIL_FROM`
  - optional `EMAIL_LOGIN_PEPPER`
