NEW API ENDPOINTS (2)
================================================================================

Endpoint 1: GET /api/wines/template
- Purpose: Download Excel import template
- Authentication: Required (Google OAuth)
- Response: Binary Excel file (template-vinhos.xlsx, ~50KB)
- Status Codes: 200 OK, 401 Unauthorized, 500 Server Error

Endpoint 2: POST /api/wines/import
- Purpose: Process uploaded Excel file and import wines
- Authentication: Required (Google OAuth)
- Request Body: { fileData: "base64_encoded_excel" }
- Response: { successful: N, failed: M, errors: [...], wines: [...] }
- Status Codes: 200 OK, 400 Bad Request, 401 Unauthorized, 500 Server Error

VALIDATION RULES IMPLEMENTED
================================================================================

Field: Nome do Vinho (Wine Name)
- Type: String
- Max Length: 160 characters
- Required: YES
- Error if missing: "Nome do Vinho é obrigatório"

Field: Produtor (Producer)
- Type: String
- Max Length: 160 characters
- Required: YES
- Error if missing: "Produtor é obrigatório"

Field: Safra (Vintage Year)
- Type: Integer
- Range: 1800 to current year + 1
- Required: NO
- Error if invalid: "Safra deve ser um número inteiro" / "Safra deve estar entre 1800 e YYYY"

Field: Tipo (Type/Grape)
- Type: String
- Max Length: 80 characters
- Required: NO
- Stored in: grape column

Field: País/Região (Country/Region)
- Type: String (split by /)
- Country Max: 80 characters
- Region Max: 120 characters
- Required: NO

Field: Preço (Price)
- Type: Decimal
- Min: 0
- Required: NO
- Error if invalid: "Preço deve ser um número válido" / "Preço não pode ser negativo"

Field: Website (Winery URL)
- Type: String (URL)
- Max Length: 300 characters
- Required: NO
- Error if invalid: "Website excede 300 caracteres"

Field: Estoque (Stock/Quantity)
- Type: Integer
- Min: 0
- Default: 1 (if not specified)
- Required: NO
- Error if invalid: "Estoque deve ser um número inteiro" / "Estoque não pode ser negativo"

Field: Notas (Notes)
- Type: String
- Max Length: 2000 characters
- Required: NO

PLAN ENFORCEMENT
================================================================================

Free Plan:
- Maximum 30 bottles total in cellar
- Import respects this limit
- If import would exceed: "Limite de garrafas gratuitas (30) seria excedido"
- User prompted to upgrade to Pro

Pro Plan:
- No bottle limit
- Unlimited import size
- Priority processing

================================================================================
ERROR HANDLING
================================================================================

File Format Errors:
- Missing fileData in request
- File must be .xlsx or .xls format
- File size must not exceed 10MB
- File must be readable Excel format

Template Structure Errors:
- Headers must match exactly (case-sensitive)
- All 9 expected columns must be present
- Valid headers: Nome do Vinho, Produtor, Safra, Tipo, País/Região, Preço, Website, Estoque, Notas

Validation Errors:
- Per-row validation with line numbers
- Character limit violations
- Type conversion failures
- Range/format violations
- Business rule violations

User-Friendly Messages:
- Clear indication of which field failed
- Why it failed (missing, invalid type, out of range, etc.)
- Line number where error occurred
- Next steps to resolve
